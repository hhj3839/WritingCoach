 작성자: 홍현진

1. 프로젝트 개요
1.1 앱 이름
가제: 글코치 AI
설명: 초등학생이 자신의 글을 스스로 수정하고 발전시킬 수 있도록 AI 기반 질문형 피드백을 제공하는 글쓰기 코칭 앱 
1.2 개발 배경
교실에서 마주한 문제점
교사가 학생 개별 글쓰기에 충분한 피드백을 제공하기 어려움
학생들은 자신의 글에서 무엇을 수정해야 하는지 스스로 파악하기 어려움
맞춤법 검사 중심 피드백에 그치는 경우가 많음
학생들이 글쓰기를 부담스럽고 어려운 활동으로 인식함
글쓰기 과정 중심 수업 운영이 어려움
기대 효과
학생 개별 피드백 시간 단축
학생 자기 수정 능력 향상
글쓰기 참여도 증가
학생의 글쓰기 자신감 향상
과정 중심 글쓰기 수업 지

1.3 목표 사용자
주 사용자: : 초등학교 3~6학년 학생 및 담임교사 
사용 환경: 국어 수업 중, 글쓰기 과제 수행 시간
기기: 태블릿 

2. 핵심 기능 정의
2.1 필수 기능 (MVP)
반드시 구현되어야 하는 최소 기능
기능 1: 맞춤법 및 문장 표현 검사 
설명: 학생이 작성한 글의 맞춤법, 띄어쓰기, 문장 반복 표현 등을 AI가 자동 분석하여 수정 방향을 안내한다. 
사용 시나리오:
학생이 글을 작성한다.
“검사하기” 버튼을 누른다.
AI가 오류 부분을 표시한다.
학생이 수정 제안을 확인하고 글을 수정한다.
우선순위: 상
기능 2: 글쓰기 유형별 피드백 
설명: 글의 종류(일기, 독후감, 설명문, 주장글 등)에 따라 다른 기준으로 피드백을 제공한다. 
사용 시나리오:
학생이 글 유형을 선택한다.
글을 작성 후 제출한다.
AI가 유형별 기준으로 글을 분석한다.
유형에 맞는 피드백을 제공한다.
우선순위: 상
기능 3: AI 질문형 코칭 피드백 
설명: AI가 정답을 제공하지 않고 질문을 통해 학생이 스스로 수정하도록 유도한다.
사용 시나리오:
학생이 AI 피드백을 확인한다.
질문형 피드백을 읽는다.
스스로 글을 수정한다.
우선순위: 

2.2 향후 추가 기능
나중에 개발할 수 있는 기능들
[글쓰기 점수 및 성장 리포트]:학생 글을 여러 요소로 분석하여 점수와 성장 변화를 시각적으로 제공한다. 
[수정 전·후 비교 기능]:학생이 어떻게 글을 수정했는지 비교하여 성장 과정을 시각적으로 보여준다.
[교사용 학급 대시보드]: 교사가 학생들의 글쓰기 활동 및 성장 데이터를 확인할 수 있다.
